Academic School Projects

Salud Integral AM

The objective of this project was to develop an Android application for elder people, which they could use to register their different daily activities and make appointments for personal events, while also promoting the health and physical activation of the user. I was mainly in charge of creating the way in which the user could pick a date and register events for such selected date. This was achieved using Caldroid and pairing the information provided by the user with the local database in which the appointments where saved.

Casa Retiro

The objective of this project was to develop a Java application which could aid the nurses at a nursing home by keeping track of the people registered at it, while also keeping track of the medicines each person needs. My contribution in this project included the assignment of new medicine to an elder person and the feature of displaying the medicine assigned to a specific one.
